# Caesar Cipher

## Overview

A Python implementation of the Caesar Cipher encryption technique. The application allows users to encode and decode messages by shifting letters through the alphabet based on a user-defined key. This project demonstrates the use of functions, parameters, and string manipulation to solve a real-world problem.

## Concepts Applied

* Functions
* Function Parameters
* Arguments
* Variables
* Lists
* Loops
* Conditional Statements (`if`, `else`)
* String Manipulation
* User Input Handling
* Modular Programming

## Learning Outcomes

* Create reusable functions with parameters
* Pass arguments between functions
* Manipulate strings programmatically
* Implement basic encryption and decryption logic
* Organize code into logical, maintainable components
* Develop problem-solving skills through algorithm design

## Features

* Message encryption
* Message decryption
* User-defined shift values
* Support for multiple operations within a single session
* Interactive command-line interface

## Technologies

* Python 3
