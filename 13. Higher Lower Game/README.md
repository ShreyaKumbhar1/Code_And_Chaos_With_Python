# Higher Lower Game

## Overview

A Python command-line game inspired by the popular "Higher Lower" format. The application presents two entities and prompts the user to guess which one has a higher follower count. The game continues until an incorrect guess is made, tracking the player's score throughout the session.

## Concepts Applied

* Functions
* Dictionaries
* Lists
* Loops
* Conditional Statements (`if`, `elif`, `else`)
* Variables
* User Input Handling
* Random Module
* Modular Programming
* Data Processing

## Learning Outcomes

* Work with complex data structures such as dictionaries and lists
* Implement game logic using functions and control flow
* Process and compare structured data
* Organize code into reusable and maintainable components
* Develop interactive command-line applications

## Features

* Random selection of comparison entities
* Score tracking system
* Continuous gameplay until an incorrect guess
* Dynamic comparison of follower counts
* Interactive command-line interface

## Technologies

* Python 3
