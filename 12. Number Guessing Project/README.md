# Number Guessing Game

## Overview

A Python command-line game where the player attempts to guess a randomly generated number within a limited number of attempts. The application provides feedback after each guess, helping the player narrow down the correct answer through logical reasoning.

## Concepts Applied

* Functions
* Local Scope
* Global Scope
* Constants
* Variables
* Conditional Statements
* Loops
* Random Module
* User Input Handling
* Control Flow

## Learning Outcomes

* Understand the difference between local and global scope
* Define and use constants effectively
* Organize program logic using functions
* Implement game mechanics with loops and conditionals
* Develop interactive command-line applications
* Apply problem-solving techniques to guide user interactions

## Features

* Random number generation
* Multiple difficulty levels
* Limited attempts system
* Hint-based feedback ("Too High" / "Too Low")
* Win and loss conditions
* Interactive command-line interface

## Technologies

* Python 3
