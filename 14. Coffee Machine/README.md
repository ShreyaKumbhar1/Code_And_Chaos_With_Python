# Coffee Machine

## Overview

A Python command-line application that simulates the functionality of a coffee vending machine. The program manages available resources, processes user orders, handles monetary transactions, and generates operational reports. This project demonstrates the integration of multiple programming concepts to build a complete interactive application.

## Concepts Applied

* Functions
* Dictionaries
* Nested Dictionaries
* Loops
* Conditional Statements (`if`, `elif`, `else`)
* Variables
* User Input Handling
* Control Flow
* Data Processing
* Code Organization

## Learning Outcomes

* Design and develop a complete command-line application
* Manage and update application state using dictionaries
* Implement resource tracking and validation logic
* Process user input and handle multiple execution paths
* Organize code into reusable and maintainable functions
* Apply problem-solving techniques to simulate real-world systems

## Features

* Multiple beverage options
* Resource availability management
* Coin-based payment processing
* Transaction validation and change calculation
* Profit tracking and reporting
* Interactive command-line interface

## Technologies

* Python 3
