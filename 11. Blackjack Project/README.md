# Blackjack

## Overview

A Python-based implementation of the classic Blackjack card game. The application simulates gameplay between a player and a computer dealer, incorporating card dealing, score calculation, game rules, and decision-making logic. This project combines multiple Python concepts to create a complete interactive application.

## Concepts Applied

* Functions
* Lists
* Loops
* Conditional Statements
* Variables
* Random Module
* User Input Handling
* Control Flow
* Code Organization
* Algorithm Design

## Learning Outcomes

* Develop a complete command-line application
* Implement complex game logic using functions and conditionals
* Manage and manipulate data using lists
* Apply problem-solving techniques to handle multiple game scenarios
* Improve code structure and modularity
* Simulate real-world game rules programmatically

## Features

* Random card dealing
* Automated dealer gameplay
* Score calculation and comparison
* Blackjack detection
* Ace value adjustment logic
* Win, lose, and draw outcomes
* Interactive command-line interface

## Technologies

* Python 3
