# Hangman

## Overview

A command-line implementation of the classic Hangman game developed in Python. The application randomly selects a word and allows the user to guess letters while tracking remaining lives. The project demonstrates the use of loops, conditionals, functions, and data structures to build an interactive game.

## Concepts Applied

* Functions
* While Loops
* For Loops
* Conditional Statements (`if`, `elif`, `else`)
* Lists
* Strings
* Variables
* User Input Handling
* Random Module
* Module Importing
* Code Organization

## Learning Outcomes

* Implement game logic using loops and conditional statements
* Work with lists to track game state
* Utilize modules to organize code effectively
* Process and validate user input
* Develop interactive command-line applications
* Apply problem-solving techniques to manage game flow

## Features

* Random word selection
* Letter-by-letter guessing
* Life tracking system
* Win and loss conditions
* Visual game progression
* Modular code structure

## Technologies

* Python 3
