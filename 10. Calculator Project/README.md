# Calculator

## Overview

A Python command-line calculator that performs basic arithmetic operations, including addition, subtraction, multiplication, and division. The application demonstrates the use of functions with return values, enabling modular and reusable code design.

## Concepts Applied

* Functions
* Functions with Outputs
* Return Statements
* Variables
* Mathematical Operations
* Conditional Statements
* Loops
* Dictionaries
* User Input Handling
* Code Reusability

## Learning Outcomes

* Create functions that return values
* Build reusable and modular code components
* Store functions as values within dictionaries
* Implement arithmetic operations programmatically
* Design interactive command-line applications
* Apply problem-solving techniques to manage application flow

## Features

* Addition, subtraction, multiplication, and division
* Continuous calculations using previous results
* User-friendly command-line interface
* Modular function-based architecture
* Efficient operation selection using dictionaries

## Technologies

* Python 3
