# Password Generator

## Overview

A Python-based password generator that creates randomized passwords according to user-defined requirements. The application generates passwords by combining letters, numbers, and special characters, demonstrating fundamental programming concepts such as iteration, data structures, and randomization.

## Concepts Applied

* For Loops
* Range Function
* Lists
* List Manipulation
* Random Module
* Variables
* User Input Handling
* String Operations

## Learning Outcomes

* Implement iterative logic using loops
* Utilize Python lists for data storage and manipulation
* Generate random values using the `random` module
* Construct dynamic strings programmatically
* Design user-driven command-line applications

## Features

* User-defined password composition
* Randomized character selection
* Support for letters, numbers, and symbols
* Dynamic password generation based on input parameters

## Technologies

* Python 3
