# Treasure Island

A text-based adventure game built in Python where players navigate through a series of choices to find hidden treasure.

## Concepts Practiced

- Conditional statements (`if`, `elif`, `else`)
- Nested conditionals
- User input handling
- Program flow control

## What I Learned

- How to create branching logic based on user decisions
- How to structure interactive command-line applications
- How nested conditions affect program execution

## Technologies

- Python 3

## Run the Project

```bash
python main.py
```
