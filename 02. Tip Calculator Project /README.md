# Tip Calculator

## Overview

A Python command-line application that calculates how much each person should contribute when splitting a bill, including a user-defined tip percentage. The project demonstrates the use of data types, arithmetic operations, and user input handling.

## Concepts Applied

* Variables
* User Input (`input()`)
* Data Types (`int`, `float`, `str`)
* Type Conversion
* Mathematical Operations
* Number Manipulation
* f-Strings
* Order of Operations (PEMDAS)

## Learning Outcomes

* Work with different Python data types
* Convert user input into numerical values
* Perform arithmetic calculations programmatically
* Format output using f-strings
* Build an interactive command-line application

## Features

* Calculates tip based on user-selected percentage
* Splits the total bill among multiple people
* Formats the final amount to two decimal places
* Interactive command-line interface

## Technologies

* Python 3
