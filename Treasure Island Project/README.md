# Treasure Island

## Overview

Treasure Island is a command-line adventure game developed in Python. The game guides users through a series of choices, where each decision influences the outcome. The project focuses on implementing conditional logic to create an interactive user experience.

## Concepts Applied

* Conditional Statements (`if`, `elif`, `else`)
* Nested Conditionals
* Logical Operators
* Control Flow
* User Input Handling
* Variables
* String Comparison

## Learning Outcomes

* Apply conditional logic to solve programming problems
* Design interactive applications with multiple execution paths
* Implement nested decision structures
* Manage user input and program flow effectively
* Improve problem-solving and logical thinking skills

## Features

* Interactive text-based gameplay
* Multiple decision paths and outcomes
* User-driven story progression
* Clear demonstration of Python control flow concepts

## Technologies

* Python 3
