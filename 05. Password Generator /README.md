# Password Generator

A Python application that generates secure and customizable passwords based on user-defined criteria, including the number of letters, symbols, and digits.

## Concepts Practiced

- Python `for` loops
- Lists
- Randomization
- List manipulation
- User input handling

## What I Learned

- How to use loops to automate repetitive tasks
- How to work with lists and access their elements
- How to generate random selections using Python's `random` module
- How to combine and shuffle data to create dynamic outputs

## Technologies

- Python 3

## Run the Project

```bash
python main.py
```
