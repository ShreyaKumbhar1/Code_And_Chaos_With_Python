# Secret Auction Program

## Overview

A Python command-line application that simulates a silent auction system. Participants can submit bids anonymously, and the program determines the highest bidder once all entries have been collected. The project demonstrates the use of dictionaries for data storage and retrieval.

## Concepts Applied

* Dictionaries
* Dictionary Operations
* Key-Value Pairs
* Nested Data Structures
* Functions
* Loops
* Conditional Statements
* User Input Handling
* Variables
* Modular Programming

## Learning Outcomes

* Store and manage data using dictionaries
* Retrieve and update values in key-value data structures
* Process multiple user inputs efficiently
* Implement logic to determine maximum values
* Organize code using functions for improved readability and maintainability

## Features

* Anonymous bid collection
* Multiple participant support
* Automatic winner determination
* Interactive command-line interface
* Efficient bid management using dictionaries

## Technologies

* Python 3
