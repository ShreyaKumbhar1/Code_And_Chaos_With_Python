# Band Name Generator

## Overview

A Python command-line application that generates a band name based on user-provided inputs. The project introduces fundamental programming concepts, including variables, user input, and string manipulation.

## Concepts Applied

* Variables
* User Input (`input()`)
* Output (`print()`)
* String Concatenation
* String Manipulation
* Basic Python Syntax

## Learning Outcomes

* Capture and process user input
* Store and retrieve data using variables
* Combine strings to generate dynamic output
* Understand the basic structure of a Python program
* Build an interactive command-line application

## Features

* Interactive user input
* Dynamic band name generation
* Simple and user-friendly command-line interface

## Technologies

* Python 3
