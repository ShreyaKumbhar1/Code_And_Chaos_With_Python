# Rock Paper Scissors

## Overview

A command-line implementation of the classic Rock Paper Scissors game developed in Python. The application enables users to play against a computer opponent that generates moves randomly, with game outcomes determined according to standard rules.

## Concepts Applied

* Lists
* Random Number Generation
* Variables
* Conditional Statements (`if`, `elif`, `else`)
* User Input Handling
* Indexing
* Control Flow

## Learning Outcomes

* Utilize Python lists to organize and access data
* Generate random selections using the `random` module
* Implement decision-making logic through conditional statements
* Process and validate user input
* Develop interactive command-line applications

## Features

* User versus computer gameplay
* Randomized computer move selection
* Outcome evaluation based on game rules
* Interactive command-line interface

## Technologies

* Python 3
