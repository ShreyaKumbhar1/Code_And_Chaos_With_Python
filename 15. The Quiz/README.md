# Quiz Game

## Overview

A Python command-line quiz application developed using Object-Oriented Programming (OOP) principles. The application presents a series of questions to the user, evaluates responses, and keeps track of the score throughout the quiz session.

## Concepts Applied

* Object-Oriented Programming (OOP)
* Classes and Objects
* Constructors (`__init__`)
* Attributes and Methods
* Encapsulation
* Loops
* Conditional Statements
* User Input Handling
* Modular Programming
* Data Processing

## Learning Outcomes

* Design and implement classes in Python
* Create and interact with objects
* Organize code using object-oriented principles
* Separate application logic into reusable modules
* Manage and process structured data efficiently
* Build scalable and maintainable applications

## Features

* Question-based interactive gameplay
* Real-time score tracking
* Automatic answer validation
* Object-oriented code architecture
* Modular and maintainable project structure

## Technologies

* Python 3
