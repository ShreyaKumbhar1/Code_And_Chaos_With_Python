# Tip Calculator

## Overview

A Python command-line application that calculates the amount each person should pay when splitting a bill, including a customizable tip percentage.

## Concepts Applied

* Variables
* User Input
* Data Types (String, Integer, Float)
* Type Conversion
* Mathematical Operations
* Number Manipulation
* f-Strings
* Order of Operations (PEMDAS)

## Learning Outcomes

* Process and validate user-provided data
* Convert data between different Python data types
* Perform arithmetic calculations programmatically
* Format numerical output for improved readability
* Develop an interactive command-line application

## Sample Output

```python
Welcome to the Tip Calculator!

What was the total bill? ₹1500
What percentage tip would you like to give? 10
How many people to split the bill? 3

Each person should pay: ₹550.00
```
